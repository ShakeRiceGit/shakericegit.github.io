【C++】マクロについてざっくりまとめてみた

C++でプログラミングをするうえでほぼほぼ使うことになるだろうと思う「マクロ」について僕なりにざっくりまとめていこうと思います。

※詳しく解説というよりかは、なんとなくで理解できるようにしているため、言葉足らずなところもあることを理解して読んでいただきたいです。

マクロとは

マクロとは簡単にいうと、文字を別の文字に置き換える命令のことを意味します。

書き方の例） 

#define マクロ名 置き換える文字

使用例） 

#include <iostream>
    
void ABC()
{
    std::cout << "print ABC" << std::endl;
}

void EDF()
{
    std::cout << "print EDF" << std::endl;
}

int main()
{
    std::cout << "マクロ定義前" << std::endl;
    
    ABC();

#define ABC EDF // "ABC"を"EDF"に置き換えるマクロを定義

    std::cout << "マクロ定義後" << std::endl;
    
    ABC();		// マクロによって置き換えられ、EDF();が実行されることになる。

    // char, stringなどの文字列は置き換わらない
    std::cout << "ABC, EDF" << std::endl;
}

出力 

マクロ定義前
print ABC
マクロ定義後
print EDF
ABC, EDF
