名前空間の重要さを知った話

今までなぜこんなに便利なものを活用していなかったのかと後悔するくらい、便利なものだということを知りました。

名前空間とは

名前空間(Namespace)とは、同じ名前のものが複数存在しないように仕分ける（名前の衝突を分ける）仕組みのことです。
簡単に言えば箱のようなものです。

UnityでC#を使ったことがある方は、一番最初にUnityで使う際におおむね必要な名前空間名が既に定義されていると思います。 

                        
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
// ↑これのこと

public class SampleScript : MonoBehaviour
{
    // Start is called before the first frame update
    void Start()
    {
        
    }

    // Update is called once per frame
    void Update()
    {
        
    }
}

定義方法

名前空間の定義方法は簡単で、"namespace"の後に名前空間名を書いてカッコでくくるだけです。 

                        
// 省略
namespace Sample
{
    public class SampleScript
    {
        
    }
}

こうすることによって「Sample」という箱の中に「SampleScript.cs」というスクリプトがあることになるわけです。
このようにして分けることが後々活用する際に便利になります。
